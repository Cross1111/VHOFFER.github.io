# Offer对比工具

## 项目简介

Offer对比工具是一款专为职场人士设计的智能薪资分析与决策助手。它能够帮助用户精准计算税后收入，综合评估工作性价比，并生成专业的对比报告，助您做出明智的职业选择。

## 核心功能

1. **精准计算税后收入**：结合薪资结构、社保公积金、城市差异，精确计算您的税后收入。
2. **综合评估工作性价比**：考虑加班、消费水平、城市宜居度等因素，全面评估工作性价比。
3. **生成可下载对比报告**：生成专业、详细的对比报告，支持PDF格式下载，便于分享和保存。
4. **多Offer对比**：支持同时对比多个Offer（最多8个），直观展示各Offer优劣势。
5. **数据可视化**：通过图表直观展示薪资构成、收入对比和综合评分。
6. **个性化建议**：基于您的输入和偏好，提供个性化的职业选择建议。

## 如何使用

1. 打开 `index.html` 文件启动工具。
2. 点击"开始对比"按钮进入对比页面。
3. 在步骤1中填写基本信息，包括公司名称、岗位、城市等。
4. 在步骤2中填写薪资结构，包括基本工资、奖金、股票期权等。
5. 在步骤3中填写工作条件，包括社保公积金、工作时间等。
6. 完成所有步骤后，系统会自动计算并展示对比结果。
7. 您可以保存或下载对比报告，便于后续查看和分享。

## 常见问题

### 税后收入如何计算？

税后收入计算考虑以下因素：
- 基本工资、奖金、补贴等税前收入总和
- 社保公积金个人缴纳部分
- 个人所得税（根据最新税法计算）
- 城市差异（不同城市的社保公积金政策不同）

### 综合评分如何计算？

综合评分由以下维度构成：
- 薪资竞争力（20分）：税后收入在同行业/同城市的排名
- 福利完善度（15分）：社保公积金缴纳比例、补充商业保险等
- 工作强度（15分）：加班频率、节假日是否正常休息
- 城市宜居性（10分）：根据房价、交通、教育资源、空气质量等数据打分
- 职业发展（15分）：公司平台、晋升空间、培训机会
- 团队氛围（10分）：通过用户评价或调研数据估算
- 个人兴趣匹配度（15分）：用户自评
